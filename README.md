# ros_auto_setup_script
Ubuntu环境下一键配置ROS开发环境（自动切换国内源，同时解决rosdep问题）
## 使用方法
打开终端，执行
```
sh ./ros_melodic_install.sh yourpassword
```
( yourpassword 替换为系统密码 )
